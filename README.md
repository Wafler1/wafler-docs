# Wafler Documentation

Official documentation source for [Wafler](https://wafler.one), a DDoS protection platform focused on real-time mitigation and privacy-oriented security.

This repository powers the docs experience at [docs.wafler.one](https://docs.wafler.one) and includes:

- Product documentation (quickstart, dashboard, enterprise guidance)
- API documentation and integration references
- Shared documentation layout and navigation

The documentation is built using the [Fumadocs](https://github.com/fuma-nama/fumadocs) framework.

## Contributing

Contributions are welcome.

1. Fork the repository.
2. Create a branch for your change.
3. Update or add documentation in the appropriate `content/docs` section.
4. Open a pull request with a clear summary of what changed and why.

## Reporting Issues

If you find incorrect information, broken links, unclear instructions, or any docs bug, please open an issue:

- [GitHub Issues](https://github.com/Wafler1/wafler-docs/issues)

Include the page/path involved, what you expected, and what happened.
